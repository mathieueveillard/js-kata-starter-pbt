# js-kata-starter-pbt

Starter project for kata with Property-Based Testing (Node, TypeScript, Jest, fast-check)

## Installation

```text
git clone https://github.com/mathieueveillard/js-kata-starter-pbt.git
cd js-kata-starter-pbt
npm install
```

## Development

```text
npm test
```
